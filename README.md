---

## 📖 项目概要 (Summary)

本项目将国家视为一个运行在物理硬件上的**巨型操作系统 (State OS)**。通过对**马克思主义国家理论**、**韦伯科层制**以及**计算机系统架构**的交叉研究，我们试图拆解国家暴力机器的底层运行逻辑。

我们不研究抽象的政治道德，我们研究**物理执行矩阵**的功耗、**根证书层**的加密协议以及**决策算法**的稳定性。

---

## 🏗 国家模型的六层架构 (The Six-Layer Architecture)

本项目所有的解剖均基于以下六层逻辑架构：

| 层级 | 逻辑定义 | 计算机映射 | 核心要素 |
| :--- | :--- | :--- | :--- |
| **L6** | **合法性根证书层** | Root Certificate | 意识形态、宗教、宪法叙事 |
| **L5** | **决策算法层** | System Kernel | 统治阶级意志、最高指令集 |
| **L4** | **传导总线层** | System Bus | 科层官僚、中层指挥链 |
| **L3** | **物理执行矩阵** | Execution Units | **基层武装单元、警察、连队** |
| **L2** | **循环供养层** | Power Supply | 财政税收、资源配给系统 |
| **L1** | **物理硬件层** | Hardware / Chassis | 领土、人口规模、基础设施 |
